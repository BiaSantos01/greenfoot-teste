Nome do jogo: Super Mario

thumbnail, tela de dialogo e pós créditos: Canva.com
- Suporta até 2 jogadores;

Wario (Jogador 1):
A = esquerda
D = direita
W = cima
S = baixo
L = soltar bomba

Mario (Jogador 2):
left = esquerda
right = direita
up = cima
down = baixo
P = soltar bomba


Objetivo: acertar 10 poderes no adiversario para ganhar.

Autores:
- Beatriz de Oliveira
